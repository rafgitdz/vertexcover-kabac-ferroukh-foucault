NAME

VertexCover - Gets a vertex cover from a randomly generated graph, or from a formatted entry  

SYNOPSIS

VertexCover {[ inputFile ] | [ -g graphType <args> ]} [ -a algorithm <args> ]  

DESCRIPTION

This program, developed during an academic project at the university of bordeaux, returns a vertex cover of a generated (or given as input) graph, using one of the algorithms described below  

OPTIONS

-g, --generate-graph
Randomly generates a graph, using one of the algorithms, according to the first parameter:

0 n p : Generates a graph with n vertices, and a probability p to draw an edge between two vertices (p must be between 0 an 1)
1 n p c : Generates a graph with n vertices, a probability p to draw an edge between two vertices (p must be between 0 an 1), and with a vertex cover with c vertices
2 n p r : Generates a bipartite graph with n vertices,a probability p to draw an edge between two vertices, and a ratio r between the two parts of a graph (p and r must be between 0 an 1)
3 n : Generates a tree with n vertices
4 n s : Generates a tree with n vertices, where each vertex has s sons
-a, --algorithm
Launches an algorithm for the generated graph or input graph,according to the first parameter

0 : Uses the greedy algorithm
1 : Uses the greedy simple algorithm
2 : Uses the 2-approached "cover edges" algorithm
3 : Uses the 2-approached algorithm based on a deep search tree
4 : Uses the optimal algorithm for bipartite graphs (for now, it can only be used with a generated graph)
5 : Uses the optimal algorithm for trees (for now, it can only be used with a generated tree)
{ 6 | 7 } s : Uses the first | second recursive algoritm, which will try to find a cover with at most s vertices (you should avoid the use of this algorithm for large covers)
8 : Uses minisat to find a cover
9 s : Uses minisat to try to find a cover with at most s vertices (you should avoid the use of this algorithm for large covers)
-dot
Generates a visual result of the vertex cover, as a png file (generated with dot)
-h, --help
prints usage and exits
-t, --test
Launches a serie of tests to ensure that each algorithm returns a vertex cover
